Incluir Archivos

Paso 2: Incluir los archivos de hey grid.

En el head del documento HTML

						<link type="text/css" rel="stylesheet" href="css/hey-grid.css">

o, dentro de tu hoja de estilo

						@import url("css/hey-grid.css");

Configuracion

Paso 3: Comprimir html, esta grid esta hecha usando el potencial de css es importante y recomendado comprimir el documento html y con esto remover espacios en blanco entre tags.

Lo puedes hacer de diferentes maneras. Segun el lenguage que uses

Django, Twig Symfony 
						{% spaceless %}
<body>
	# Code 	
</body>
{% endspaceless %}
PHP: 
					function spaceless($buf){
	return str_replace(array("\n","\r","\t"),'',$buf);
}
html - Javascript: Despues del tag body 
						<script>document.body.innerHTML = document.body.innerHTML.replace(/>\s+</g, "><");</script>

Una Prueba Rapida

Paso 3: Probar que todo vaya bien.

Si ya hiciste los pasos anteriores y quieres saber rapidamente si la Grid esta bien configurada, solo es que peges el siguiente html despues del tag <body> .

HTML 
					<section class="grid-columns" heygrid-debug>
	<div class="cols-4"></div>
	<div class="cols-8"></div>
</section>
